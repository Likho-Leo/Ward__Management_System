# Ward Management System

##  Project Overview

**Ward Management System** is a web-based hospital management application built using **ASP.NET Core MVC** and **Dapper**.  
The system follows a layered architecture and implements **role-based access control** to manage ward operations efficiently.

This project was developed as part of a group academic project, where I was responsible for implementing the **Nurse** and **Nursing Sister** subsystems.

---

##  My Contribution

### I designed and implemented the following subsystems:

- Nurse management module  
- Nursing Sister management module  
- Patient record handling within assigned roles  
- Role-based authorization logic  
- CRUD operations using Dapper data access  
- Form validation and secure data submission  
- Controller-level access restrictions  

The subsystems were developed using the **MVC architectural pattern** with structured separation between:

- Presentation Layer  
- Business Logic Layer  
- Data Access Layer  

---

##  System Architecture

The application follows a layered design:

- **Presentation Layer** – ASP.NET Core MVC Views and Controllers  
- **Repository Layer** – Data abstraction using repository pattern  
- **Data Access Layer** – Dapper-based SQL execution  
- **Database Layer** – SQL Server relational storage  

Authentication is implemented using **cookie-based authorization** with role validation.
